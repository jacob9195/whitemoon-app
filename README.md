# 오늘의 사주

> 생년월일을 입력하면 사주팔자와 성향을 쉬운 한국어로 풀어주는 웹앱이에요.  
> 회원가입 없이 바로 사용할 수 있어요.

---

## 화면 구성

이 앱은 세 개의 화면으로 이루어져 있어요.

```
홈 화면 (/)
  └─ "내 사주 보기" 버튼
        ↓
입력 화면 (/input)
  └─ 생년월일 / 양력·음력 / 출생시간 / 성별 입력
        ↓ (유효성 검사 통과 시)
결과 화면 (/result)
  └─ 사주팔자 / 오행 분포 / 성향 요약 / 직업·연애·주의점
```

### 홈 화면 `/`
- 앱 소개 문구와 시작 버튼이 있어요.
- "내 사주 보기" 버튼을 누르면 입력 화면으로 이동해요.

### 입력 화면 `/input`
- **양력/음력** — 기본값은 양력이에요. 모르면 양력을 선택하세요.
- **생년월일** — 년도(4자리), 월, 일을 각각 입력해요.
- **출생 시간** — 시간을 모를 경우 "모름" 버튼을 누르면 시간 없이도 결과를 볼 수 있어요.
- **성별** — 남성 / 여성 중 선택해요.
- 입력을 빠뜨리거나 잘못 입력하면 빨간 안내 메시지가 나타나요.

### 결과 화면 `/result`
- **사주팔자** — 연주·월주·일주·시주 4개의 기둥을 한자와 한글 설명으로 보여줘요.
- **오행 분포** — 목·화·토·금·수 기운의 비율을 막대 차트로 표시해요.
- **성향 요약** — 일간(나를 나타내는 글자)을 기준으로 핵심 성격을 설명해요.
- **인생 가이드** — 직업 / 연애 / 주의점 탭으로 나눠 보여줘요.
- ⚠️ 현재 "임시 데이터"가 표시돼요. 사주 계산 모듈이 연결되면 실제 결과로 바뀌어요.

## 처음 실행하는 방법

### 1단계 — Node.js 확인

터미널(맥: Terminal 앱 / 윈도우: PowerShell)을 열고 아래를 입력하세요.

```
node -v
```

`v18` 이상이 나오면 OK예요. 없으면 https://nodejs.org 에서 설치하세요.

### 2단계 — 프로젝트 폴더로 이동

```
cd saju-app
```

### 3단계 — 패키지 설치 (처음 한 번만)

```
npm install
```

### 4단계 — 개발 서버 실행

```
npm run dev
```

브라우저에서 http://localhost:3000 을 열면 앱이 보여요.  
코드를 수정하면 저장과 동시에 화면이 자동으로 새로고침돼요.

## Vercel에 배포하는 방법 (인터넷에 올리기)

1. https://github.com 에 프로젝트를 올려요 (저장소 만들기 → 코드 업로드)
2. https://vercel.com 에 GitHub 계정으로 가입해요
3. "New Project" → GitHub 저장소 선택 → "Deploy" 클릭
4. 1~2분 뒤 자동으로 배포돼요
5. 이후 코드를 수정해서 GitHub에 올리면 자동으로 다시 배포돼요

## 자주 묻는 질문

**Q. "임시 데이터" 배지는 언제 사라지나요?**  
A. 사주 계산 모듈(`lib/saju/calculator.ts`)이 완성되고 결과 페이지에서 연결하면 없어져요.

**Q. 결과 URL을 직접 열면 어떻게 되나요?**  
A. 입력값이 없으면 자동으로 입력 화면으로 돌아가요.

**Q. 배포 후 코드를 수정하면 어떻게 되나요?**  
A. GitHub에 올리면 Vercel이 자동으로 새 버전을 배포해줘요.

**Q. API 키는 어디에 넣나요?**  
A. `.env.local.example` 파일을 복사해서 `.env.local`로 이름을 바꾼 뒤, 값을 채워 넣으세요.  
이 파일은 GitHub에 절대 올리면 안 돼요 (.gitignore에 이미 포함돼 있어요).
